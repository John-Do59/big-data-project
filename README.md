# Big Data Pipeline - Olist (PySpark)

##  Contexte
Projet de pipeline Big Data basé sur le dataset Olist (e-commerce brésilien).

##  Objectif
Construire une pipeline PySpark organisée en :
- raw
- bronze
- silver
- gold

##  Stack
- PySpark
- Python
- Parquet
- Git workflow (feature → develop → main)

##  Pipeline
RAW → BRONZE → SILVER → GOLD

##  Équipe
Projet réalisé en équipe de 3 développeurs Data/IA.
